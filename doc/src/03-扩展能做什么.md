# 为什么要学习扩展？
1. 培养写扩展的意识。一个人如果不具备写扩展的能力，当遇到问题的时候，他比较难以想到实用扩展来解决问题。  
2. VsCode的扩展非常简单。VsCode提供了非常易用的创建扩展的方法，看文档半天就能够上手开发。  


# 定制主题

2. 图标样式
